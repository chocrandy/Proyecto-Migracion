Migracion de Guatemala

Es un sistema que permita agilizar el trámite para la emisión de pasaportes. El cual consiste en llenar un formulario digital y en un plazo de 24 horas recibirá un correo donde indica la hora y el día en la que el usuario debe presentarse para realizar el trámite.

Sitio web Para solicitudes

Para poder realizar las verificaciones se tienen que realizar desde el portal web y ingresar los datos necesarios para que la administración pueda gestionar el proceso de solicitud de pasaporte.
